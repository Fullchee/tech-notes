### Writing mode

flex direction will depend on whether your text is left/right or right/left or top/down

### `flex` shorthand

-   Default
    -   flex-grow: 0
    -   flex-shrink: 1
    -   flex-basis: auto
-   `flex: 1`
    -   flex-grow: 1
    -   flex-shrink: 0
    -   flex-basis: 0

### `place-items: center stretch`
